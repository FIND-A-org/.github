## About FIND-A

**FIND-A**는 **Financial Data Analysis**의 약자로,  
금융시장과 데이터 분석, AI, 투자전략, 리스크 관리 등 디지털 금융 분야를 함께 탐구하는 금융데이터분석학회입니다.

단순한 이론 학습을 넘어, 실제 금융 데이터를 기반으로 문제를 정의하고 분석하며,  
데이터 기반 의사결정 역량을 갖춘 실무형 인재 양성을 지향합니다.
